Visualizar Wikis

Descripción

Se permite a todos los usuarios visualizar la lista de wikis disponibles en el sistema.

CU-4

Nombre

Modificar contenido de artículos

Descripción

Se permitirá modificar el contenido

Actores

Colaborador, Supervisor

Precondiciones

Haber iniciado sesión con una cuenta registrada

Flujo Básico

  1. El usuario descargará un archivo HTML para editar el artículo
  2. El usuario subirá el archivo como propuesta

Flujo Alternativo

  1. Si el usuario que editará es un supervisor no se enviará la notificación, se hará el cambio directamente

Poscondiciones

Se enviará una notificación al supervisor avisando que hay una propuesta de cambio en un artículo
